# AI-Ransomware-Hunter-Pro-v4.5
Project Definition: "AI Ransomware Hunter Pro v4.5" is a real-time cybersecurity dashboard combining machine learning with low-level system auditing.

RAM Monitoring: It actively sniffs volatile memory (RAM) to instantly flag and isolate malicious process behaviors before execution.

Live Mitigation: The system features an interactive HUD allowing operators to kill threats immediately via native OS-level SIGKILL commands.

Storage Scanning: It includes a Static Directory Hunter that uses mathematical entropy analysis to detect ransomware encryption on the hard drive.

Development Value: Built in a 3-day sprint, this project serves as a fully functional, responsive local EDR prototype for advanced threat hunting.
